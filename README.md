# Calculator App

A simple calculator web application for basic arithmetic operations, built using JavaScript, HTML, and CSS.

## Features
- Addition, subtraction, multiplication, division
- Decimal calculations
- Clear functionality

## Technologies Used
- JavaScript
- HTML5
- CSS3

## How to Use
1. Open `index.html` in your browser.
2. Click buttons to input numbers and operators.
3. Press "=" to calculate.
4. Press "C" to clear the display.
